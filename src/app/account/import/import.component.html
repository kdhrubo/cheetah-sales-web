Upload Notes

The uploaded file should a comma , separated text file ending with .csv extention.

The maximum file size allowed is 1MB.

The fields should be in the order below

  1. Name
  2. Website
  3. Phone
  4. Fax
  5. AnnualRevenue
  6. NoOfEmployees
  7. Company
  8. Email
  9. Mobile

In case of missing fields, just leave it blank. Refer to the example line below where company, country and email fields are missing.

John,www.example.com,93456899,93456899Fax,93456899,,,mobile
