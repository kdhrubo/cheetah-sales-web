Upload Notes

The uploaded file should a comma , separated text file ending with .csv extention.

The maximum file size allowed is 1MB.

The fields should be in the order below

  1. First Name
  2. Last Name
  3. Email
  4. Company
  5. Country
  6. City
  7. Designation
  8. Mobile
  9. LinkedIn
  10. Facebook
  11. Twitter

In case of missing fields, just leave it blank. Refer to the example line below where company, country and city fields are missing.

John,Stones,jstones@example.com,MicroFocus, ,,,93456899,myLinkedIn,myfacebook,mytwitter
